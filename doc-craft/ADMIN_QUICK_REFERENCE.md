# Admin Panel Quick Reference

## Navigation
- Dashboard: `/admin`
- Homepage Products: `/admin/homepage`
- Hero Sections: `/admin/hero-sections`
- Products: `/admin/products`
- Orders: `/admin/ecommerce/orders`
- Shipping: `/admin/shipping`
- Categories: `/admin/categories`
- Landing Pages: `/admin/landing-pages`
- Media: `/admin/media`

## Homepage Products (`/admin/homepage`)
**Purpose**: Assign products to homepage sections

**Sections** (4 slots each):
- Hero & Featured
- Best Sellers
- New Arrivals
- On Sale

**Actions**:
- Select products from dropdowns
- Clear section
- Save changes

## Hero Sections (`/admin/hero-sections`)
**Purpose**: Manage homepage carousel slides

**Actions**:
- Add/Edit/Delete slides
- Toggle active/inactive
- Reorder slides
- Set images, text, CTAs

**Key Fields**:
- Title, Subtitle, Description
- CTA Text & Link
- Background (gradient/image/video)
- Card Image/Video
- Display Order, Active Status

## Quick Tips
- Homepage Products: Products can only be in ONE section
- Hero Sections: Deactivate instead of delete
- Changes require clicking Save
- Refresh to see latest data
