# Admin Homepage Management

## Location
`/admin/homepage`

## Purpose
Curate which products appear in homepage sections (Hero, Best Sellers, New Arrivals, On Sale).

## How It Works
1. **4 Sections** - Each has 4 product slots
2. **Drag & Select** - Choose products from dropdown for each slot
3. **Visual Preview** - See product images and prices
4. **Save Changes** - Click Save to publish (shows count of changes)

## Sections
- **Hero & Featured** - Main carousel products
- **Best Sellers** - Customer favorites
- **New Arrivals** - New products
- **On Sale** - Discounted items

## Quick Actions
- **Clear** - Remove all products from a section
- **Refresh** - Reload product list
- **Save** - Publish changes (only enabled when changes exist)

## Notes
- Products can only appear in ONE section at a time
- Changes are immediate after saving
- Empty slots show "Empty" placeholder
